The LXQt team announces the release of lxqt-session 2.3.0.
The release can be downloaded from [Github](https://github.com/lxqt/lxqt-session/releases).

Main changes:

 * Showed autostart applications in the correct category.
 * Removed "Experimental" from Wayland settings page.
 * Don't set (X11) global scale factors under Wayland.
 * Removed colons from two groupbox titles in `lxqt-config-session`.

<br/>
A full list of changes is in the CHANGELOG file.
