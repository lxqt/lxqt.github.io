The LXQt team announces the release of pcmanfm-qt 2.4.0.
The release can be downloaded from [Github](https://github.com/lxqt/pcmanfm-qt/releases).

Main changes:

 * Removed redundant spacing for filter bar.
 * Removed redundant .ui <hints> tags.
 * Consistent visibility of multi-screen desktop items on Wayland.

<br/>
A full list of changes is in the CHANGELOG file.
