The LXQt team announces the release of lxqt-powermanagement 2.3.0.
The release can be downloaded from [Github](https://github.com/lxqt/lxqt-powermanagement/releases).

Main changes:

 * Added a section for turning off monitor on Wayland (supporting KWin, Niri and Hyprland).
 * Corrected the tabbing order in the config dialog.
 * Added Apply button to the config dialog.
 * Added tooltip info about idleness on fullscreen on Wayland.

 <br/>
A full list of changes is in the CHANGELOG file.
