The LXQt team announces the release of lxqt-powermanagement 2.4.0.
The release can be downloaded from [Github](https://github.com/lxqt/lxqt-powermanagement/releases).

Main changes:

 * Separated idleness options for turning off monitor(s)
 * UI improvement for Battery/AC idleness settings.
 * Removed useless options.
 * Updated translations.
 * Small UI improvements.

 <br/>
A full list of changes is in the CHANGELOG file.
