The LXQt team announces the release of lxqt-notificationd 2.3.0.
The release can be downloaded from [Github](https://github.com/lxqt/lxqt-notificationd/releases).

Main changes:

 * Set the minimum allowed width to 100px.
 * Fixed keeping notifications open by putting the mouse cursor over their windows.
 * Corrected the tabbing order in the config dialog.


<br/>
A full list of changes is in the CHANGELOG file.
