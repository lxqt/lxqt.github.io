The LXQt team announces the release of lxqt-qtplugin 2.3.0.
The release can be downloaded from [Github](https://github.com/lxqt/lxqt-qtplugin/releases).

Changes:

 * Changed default cursor size from 16 to 24.
 * Fixed building with Qt 6.10.

A full list of changes is in the CHANGELOG file.
