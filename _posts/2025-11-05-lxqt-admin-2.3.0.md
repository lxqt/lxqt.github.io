The LXQt team announces the release of lxqt-admin 2.3.0.
The release can be downloaded from [Github](https://github.com/lxqt/lxqt-admin/releases).

Main changes:

 * Prevented Delete button triggering with no selection.
 * Do not prompt for new user password after cancelled auth.
 * Show it in menus outside LXQt too.

<br/>
A full list of changes is in the CHANGELOG file.
