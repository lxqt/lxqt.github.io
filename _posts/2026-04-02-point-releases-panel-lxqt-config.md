The LXQt team announces point releases of **lxqt-config 2.3.2** and **lxqt-panel 2.3.3**.


Changes in both:

 * Fixed too strict parsing of icon themes with Qt6.11, which caused empty icon themes settings.

<br/>

Changes in lxqt-panel:
 * Prioritize starting strings of titles with Fancy Menu's filtering.
 * Disabled shortcutselector in panel menus on Wayland and show tooltip information.
 * Qt 6.10 version check for filterstring.
 * Removed obsolete Qt5 version check.
 * Fixed auto-hiding on dragging over Task Manager and Show Desktop.
 * KeyboardInteractivityNone also for labwc (0.9.4).
 * Added tooltip to opacity slider to communicate value.
 * Cleaned up some ui files.
 * Updated translations and code cleanup.
  
<br/>
The releases can be downloaded from Github:
 [lxqt-panel](https://github.com/lxqt/lxqt-panel/releases), [lxqt-config](https://github.com/lxqt/lxqt-config/releases).
