The LXQt team announces the release of libqtxdg 4.3.0.
The release can be downloaded from [Github](https://github.com/lxqt/libqtxdg/releases).

Changes:

 * Fixed building with Qt 6.10.
 * Fixed xdgdirs' `createdir` behavior.
 * Miscellaneous code improvements.
 * Xdg dirs better postfix
 * Fixed XdgDirs documentation typo.



<br/>
A full list of changes is in the CHANGELOG file.
