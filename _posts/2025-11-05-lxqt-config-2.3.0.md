The LXQt team announces the release of lxqt-config 2.3.0.
The release can be downloaded from [Github](https://github.com/lxqt/lxqt-config/releases).

Main changes:

 * Changed the default cursor size from 16 to 24.
 * Fixed "Open theme folder" context menu showing on empty space.
 * Silenced several compilation warnings.
 * Removed unneeded QOverloads.
 * Improved autostarts.
 * Added missing quotes to autostart *.yaml files.



<br/>
A full list of changes is in the CHANGELOG file.
