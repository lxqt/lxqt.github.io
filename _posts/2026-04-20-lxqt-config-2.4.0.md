The LXQt team announces the release of lxqt-config 2.4.0.
The release can be downloaded from [Github](https://github.com/lxqt/lxqt-config/releases).

Main changes:

 * Use std::pair instead of QPair.
 * Added 'ms' to slider labels, missing page title and removed the redundant labels in keyboard repeat settings.
 * Don't show LXDE settings.



<br/>
A full list of changes is in the CHANGELOG file.
