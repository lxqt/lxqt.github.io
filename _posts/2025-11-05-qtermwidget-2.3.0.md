The LXQt team announces the release of qtermwidget 2.3.0.
The release can be downloaded from [Github](https://github.com/lxqt/qtermwidget/releases).

Changes:

 * Support emoji flags.
 * Moved away from deprecated declarations in `pyproject.toml`.
 * Updated the example for using PyQt6.

<br/>
A full list of changes is in the CHANGELOG file.
