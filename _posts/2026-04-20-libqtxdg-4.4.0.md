The LXQt team announces the release of libqtxdg 4.4.0.
The release can be downloaded from [Github](https://github.com/lxqt/libqtxdg/releases).

Changes:

 * Modernized string nomenclature and classes.
 * Fixed compilation with Qt ≥ 6.10.
 * Prefer std:: functions.
 * Added $XDG_STATE_HOME directory implementation.
 * Fixed NotShowIn=LXQt handling in .desktop files.



<br/>
A full list of changes is in the CHANGELOG file.
