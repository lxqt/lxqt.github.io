A light Qt desktop environment.

The focused classic with a modern look and feel. An unhindered and clear way to go.

  • Modular components
  • Powerful file manager
  • Customize appearance everywhere
  • Panel(s) with many plugins and settings
  • Window Manager agnostic

Applications

  • lxqt-runner — Launch applications and do calculations
  • PCManFM-Qt — Browse files
  • LXImage-Qt — View images
  • QTerminal — Use the terminal
  • ScreenGrab — Capture screenshots
  • LXQt Archiver — Manage archives
  • Qps — Manage processes

Try LXQt in most Linux or BSD distributions.
Learn to install it on the LXQt wiki.

Read more about its history in the LXQt wiki.
