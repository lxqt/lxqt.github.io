Releases

Latest version is LXQt 1.4.0, released in November 2023.

All releases are hosted on GitHub; new versions are released usually in April and November.

In the LXQt GitHub organization you can find all releases for each individual component - for example lxqt/libfm-qt/releases lists all the releases for libfm-qt. There can be downloaded libfm-qt-VERSION.tar.xz and libfm-qt-VERSION.tar.xz.asc.

Distributions

Almost all linux distributions and FreeBSD include precompiled binary packages for LXQt. An overview about distributions and relative versions is avalaible in our Wiki.
